# fire - Deteção de incêndio - Arduino

Este repositório contém o código *Arduino* desenvolvida no âmbito de uma unidade curricular designada **Redes Móveis e Sem Fios**.

O objetivo é construir um dispositivo capaz de detetar e extinguir um incêndio num local fechado. 

Para isto são utilizados sensores de temperatura e humidade (DHT11) e um de qualidade do ar (MQ 135) e os seguintes atuadores: mini-buzzer e bomba de água. o dispositivo comunica com um servidor desenvolvido em *Python* através da tecnologia *Sig Fox* que é uma tecnologia *Low-Power* dedicada a este tipo de aplicações.

Nota de utilização: é necessário importar uma biblioteca cujasinstruções se encontram [aqui](https://www.instructables.com/id/How-to-interface-Humidity-and-Temperature-DTH11-Se/)

Associado a este repositório estão outros 2 que contém o servidor web desenvolvido em *Python* e uma aplicação *mobile* relacionados com o projeto
